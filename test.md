# 新数据结构构思(给数组使用)
## typeof index == uint32_t

### 结构 Hash Bitmaped Blocked Skip List
### HBBSL 用于管理分段密集数组/纯密集数组形式
### 一种KV结构,但可以像数组一样遍历
- 将index视为 多个段左高右低[12][16][4]
- 通常level0应该只有极少节点(毕竟是高12位),

### 结构
- 12bits Hash<Node> 开放地址法哈希表,布谷鸟哈希
- SkipList<Node> 跳表,层高最大16
- Array<Data> 16个块,对齐索引

### 示意图
```
L0:     [0]
         |
L1:     [0]   [8]    [16]
        [0][4][8][12][16]
         |  |  |  |   |
L2:     [    0 - 196    ]
```
访问元素 = 1 次hash + log(N) 次 跳表索引 + 1次偏移直接取
